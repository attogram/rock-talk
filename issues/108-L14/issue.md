Issue: Compressed Rock Scaling & Format Collapse Study

Problem Statement

Rock Talk currently has multiple representation forms in circulation, but the system is converging toward two primary operational formats:

1. Full Rock (max signal, structured human-readable compression)
2. Compressed Rock (multi-scale token budgets)

All other formats (prose, hybrid, schema variants) are considered non-primary or transitional artifacts.

The key unresolved question is how semantic integrity behaves under aggressive compression.

---

Core Hypothesis

Rock Talk is not format-dependent.

It is compression-invariant up to a threshold, after which:

- semantic loss increases non-linearly
- CDI instability rises
- agent agreement becomes artificially inflated or collapsed

We hypothesize:

«There exists a compression curve where intent preservation remains stable until a phase transition point.»

---

Compression Tiers (Proposed Standard)

65K Tier — Full Context Compression

- Near-complete representation
- Includes:
  - full structure
  - all arguments
  - full reasoning graph
- Use case:
  - full paper ingestion
  - multi-agent consensus systems
  - full research loops

---

32K Tier — Research Operational Compression

- Moderate pruning of redundancy
- Keeps:
  - core claims
  - structural logic
  - CDI-relevant disagreement points
- Removes:
  - narrative scaffolding
  - extended justification chains

Use case:

- agentic analysis
- iterative refinement
- synthesis tasks

---

Ultra-Compressed Rock (≤8K–4K target class)

- Maximum density
- Only:
  - payload
  - claims
  - structural operators
- No explanatory padding
- Assumes shared ontology / prior context

Use case:

- GitHub issue bootstraps
- agent handoffs
- prompt seeding

---

Research Questions

RQ1: Compression Stability

At what compression level does:

- intent preservation break
- CDI become unstable
- agreement become misleading

---

RQ2: Phase Transition Detection

Is there a measurable threshold where:

«compression stops being lossless-in-practice and becomes lossy-in-meaning?»

---

RQ3: Task Sensitivity

Which tasks degrade first under compression?

- reasoning chains
- conceptual synthesis
- disagreement detection (CDI)
- procedural execution

---

Proposed Metrics

Intent Retention Ratio (IRR)

IRR = preserved semantic units / original semantic units

---

Compression Failure Index (CFI)

CFI = (CDI distortion + task failure rate) / compression level

---

Phase Transition Point (PTP)

Defined as:

- steep drop in IRR
- sharp rise in CFI
- divergence collapse or artificial consensus

---

Working Claim

Rock Talk is not primarily a communication protocol.

It is:

«a controllable semantic compression system with measurable failure boundaries.»

---

Implication

If validated:

- “best format” becomes task-dependent compression tier selection
- representation becomes a tunable runtime parameter
- research artifacts become multi-resolution objects
